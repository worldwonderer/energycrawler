# 本地原生环境管理

本文档描述不使用 `uv` 时的最小运行方式。

## 1. 创建虚拟环境

```bash
cd EnergyCrawler
python -m venv venv
```

激活环境：

```bash
# macOS / Linux
source venv/bin/activate

# Windows
venv\Scripts\activate
```

## 2. 安装依赖

```bash
pip install -r requirements.txt
```

## 3. 启动 Energy 服务

```bash
bash energy-service/start-macos.sh
```

## 4. 运行爬虫

```bash
# xhs
python main.py --platform xhs --lt qrcode --type search --keywords 编程副业

# x
python main.py --platform x --lt cookie --type search --keywords "open source"
```

## 5. 运行 WebUI

```bash
python -m uvicorn api.main:app --port 8080 --reload
```

说明：当前仅支持 `xhs` 与 `x`。
